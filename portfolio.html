Trans

Mobile Translation app made with Machine Learning and Google API technologies. Supports many input methods and can translate to over 40 languages.

Trans is a project developed by my a team of 3 developers including me. While my team and I moved to another part of the country for an undergraduate program, we realized a change in language, something we couldn't understand. This made life less convinient for us as we couldn't really communicate with the people over there. Realizing we weren't the only people faced with this problem, we developed a fully-functional mobile application that helped us with communication. This solution grew to be used by hundreds of students in campus.

Splash Screen and App Intro

The application has a text-translation feature, you could simply type in text in another language and see it translated in real time.

This section had to do with some machine learning algorithms, we had to implement handwriting detection and translation so we could simply scan the notes of our friends and have it translated to another language.

Using the YOLO V3 object detection algorithm, we implemented a real-time object detection and translation feature. This helped us to buy groceries/items from stores since we could easily scan - recognize - translate the product names to something both we and the seller would understand.

Speech Translation Feature, a user could record an audio and have it's text translated to another language.

File Translation, simply upload a document in a language and download the same format, but this time, with the text translated to a destination language.
